# Clove Dental Landing Page

A responsive landing page built from a Figma design using **HTML**, **CSS**, and **Vanilla JavaScript**.  
This page showcases root canal treatments, patient testimonials, Google reviews, and a lead capture form.

## Features

- Responsive layout for mobile and desktop
- Hero section with background image and doctor info
- Lead capture form with:
  - Mobile number validation
  - Random CAPTCHA generation
- Treatments section with pricing
- Testimonials section
- Google reviews cards
- Footer with links and credits

## Tech Stack

- HTML5
- CSS3 (Flexbox, media queries)
- Vanilla JavaScript (form validation + captcha)
